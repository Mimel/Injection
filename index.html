Inject.ion Reference Manual

Lexemes

Device: @[string] (e.g. @persephone)

Store: $

Field: #[string] (e.g. #cache)

Value: [store | field]

Comparator: [! | = | < | >] ('not equal to', 'equal to', 'less than', and 'greater than', respectively)

Connection Commands

Inject

Syntax: 'inject [device] [value]'

Transfers control to a connected device, passing a parameter as storage. If the parameter is the store itself, then the store does not change.

Return

Syntax: 'return [value]'

Returns control to the parent of the current device, passing a parameter as storage. If the parameter is the store itself, then the store does not change.

Data Manipulation Commands

Set

Syntax: 'set [field] [value]'

Sets a field in the current device to the value given.

Math Commands

Add

Syntax: 'add [field]'

Adds the value of a given field to the current store.

Sub

Syntax: 'sub [field]'

Subtracts the value of a given field from the current store.

Invert

Syntax: 'invert'

Inverts the current store, switching the value from positive to negative (or vice versa).

Program Flow Commands

Mark

Syntax: 'mark [string]'

Creates a point where the program can jump to, named by the given string.

Jump

Syntax: 'jump [value] [comparator] [value] [string]'

Jumps to the given previously-instantiated marker if the conditional ([value] [comparator] [value]) is true.

Skip

Syntax: 'skip [value] [comparator] [value] [integer]'

Skips a number of lines after this statement equal to [integer] if the conditional ([value] [comparator] [value]) is true.
